## <img src="https://raw.githubusercontent.com/rosgear/rg-be-references-desk/refs/heads/main/assets/images/icon.svg" width="64px" height="64px" align="absmiddle"> Расширение модуля «Панель справочников»

Панель справочников отображает все доступные пользователю справочники, установленные в системе, и обеспечивает возможность навигации между ними.

Расширение модуля входит в состав Панели управления GePanel и cлужит расширением для [модуля «Справочники»](https://github.com/rosgear/rg-be-reference).

### Установка

Для добавления расширения модуля в ваш проект, вы можете просто выполнить команду ниже:

```
$ composer require rosgear/rg-be-references-desk
```

или добавить в файл composer.json вашего проекта:
```
"require": {
    "rosgear/rg-be-references-desk": "*"
}
```
или скачать архив на [странице расширения модуля](https://rosgear.ru/component/rg-be-references-desk/) в каталоге приложений RosGear.

После добавления расширения модуля в проект выполните его установку в редакцию веб‑приложения с помощью Панели управления GePanel.

<a name="english"></a>
## <img src="https://raw.githubusercontent.com/rosgear/rg-be-references-desk/refs/heads/main/assets/images/icon.svg" width="64px" height="64px" align="absmiddle"> Module extension «References Panel»

The reference panel displays all reference installed in the system available to the user and provides the ability to navigate between them.

The module extension is part of the GePanel Control Panel and serves as an extension for the [«References» module](https://github.com/rosgear/rg-be-reference).

### Installation

To add the module extension to your project, you can simply run the command below:

```
$ composer require rosgear/rg-be-references-desk
```

or add to your project's composer.json file:
```
"require": {
    "rosgear/rg-be-references-desk": "*"
}
```
or download the archive from the [module extension page](https://rosgear.ru/component/rg-be-references-desk/) in the RosGear application catalog.

After adding the module extension to the project, install it into the web application edition using the GePanel Control Panel.
